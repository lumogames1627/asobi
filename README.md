# asobi. — 大学生限定 遊び友達マッチングアプリ（プロトタイプ）

## アプリの内容
大学生限定の「予定参加型」遊び友達マッチングアプリ。人をスワイプして選ぶのではなく、飲み・カラオケ・ごはんなどの「予定」に参加し、3人揃うとグループチャットが自動で開いて実際に会える。

## 使用言語
プロトタイプ: HTML / CSS / JavaScript（単一ファイル、外部フレームワーク不使用）。
本実装はFlutter（Dart）+ Firebase を予定。

## 工夫した内容
「人でなく予定に参加する」設計で品定め感と1対1接触を排除し、3人グループ制・年齢/性別の構造的な安全ゲート・カテゴリと理由が必須の通報フローで安全性を担保。さらに登録時の「好きなカテゴリ」を実際の並び替え・おすすめ表示・メンバーとの共通の趣味表示に反映させ、収集した個人データを機能として使い切る設計にした。

## 動かし方
`asobi_prototype.html` をブラウザで直接開くだけで動作します（外部ライブラリはCDN経由で読み込み）。

---
※ このリポジトリはUIプロトタイプ（デモ用ダミーデータ）です。認証・決済・通知等のバックエンド機能は実装されていません。
